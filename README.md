# Portfolio Website for GitHub Pages

This repository is now set up as a static portfolio site.

## Quick Start

1. Add your files into these folders:
   - `assets/images/profile.jpg` for your profile photo
   - `assets/docs/Resume 2026-01-10.docx` for your CV
   - Optional project PDFs and media into `assets/docs/` and `assets/media/`
2. Commit and push to `main`.
3. In GitHub repository settings, open Pages:
   - Source: Deploy from a branch
   - Branch: `main` and root (`/`)
4. Wait a few minutes for deployment and open your site URL.

## Edit Content

- Main content: `index.html`
- Styles: `styles.css`
- Interactions: `script.js`

## Notes

- The CV button currently points to `assets/docs/Resume 2026-01-10.docx`.
- If you prefer a PDF CV link, upload a PDF and update that href in `index.html`.
